### Напишите программу, которая преобразует 1 в 31. Для выполнения поставленной задачи необходимо обязательно и только один раз использовать:
#### Цикл for
#### *= 5
#### += 1
### Никаких других действий или циклов использовать нельзя.

```python
value = 1
for i in range(7):
    if value != 6:
        value += 1
    elif value == 6:
        value *= 5
    else:
        break
print(value)
```

Цикл for повторяется 7 раз. 
Пока value не равен 6, он будет увеличиваться на 1. 
Когда value будет равен 6, он умножается на 5 и получается 30. 
После чего value снова увеличивается на 1 и становится равен 31му. 
В конце, если value не удовлетворяет предыдущим условиям, то происходит выход из цикла
В целом, получается немного излишне, тк в итоге на последнем шаге цикла идет последнее добавление единицы и выход из цикла
